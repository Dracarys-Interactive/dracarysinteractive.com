About Dracarys Interactive

Dracarys Interactive was founded by John Sissler. John has been an Adjunct Professor of Computer Science at Sussex County Community College since 2019 concentrating on Game Development using the Unity Game Engine. Dracarys Interactive specializes in creating custom interactive 2D, 3D, and XR software with a focus on research-driven open-source projects.
